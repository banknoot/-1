مشروع جاهز للنشر على Vercel — نسخة صفحة واحدة
---------------------------------------------
تعليمات سريعة للنشر:

1) فك الضغط وارفع المجلد 'stock-ai-vercel' إلى مستودع GitHub جديد.
2) سجّل دخولك إلى https://vercel.com وادخل عبر GitHub.
3) اضغط "New Project" واختر المستودع الذي رفعته.
4) اقبل الإعدادات الافتراضية واضغط "Deploy".

بعد النشر سيصبح الموقع على شكل: https://<your-project>.vercel.app
